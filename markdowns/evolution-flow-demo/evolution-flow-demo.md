# [주요 기능]{1} 프레젠터

![](05-주요기능-a.png)

- 프레젠터로 발표합니다.
- [F]를 누르면 전체화면이 됩니다.
- [좌], [우] 키로 슬라이드를 이동합니다.
- Remotion의 애니메이션 효과를 프레젠터에서 사용할 수 있습니다.

---

# [주요 기능] 에디터

- 에디터 상태에서 수정할 수 있습니다.
- 템플릿 규칙(스킬)에 의해 사용해야 합니다.
- 예를 들어 이미지를 허용하는 슬라이드 템플릿에서만 이미지를 넣을 수 있습니다.

---

# [이렇게 사용하세요] 추천 워크 프로세스

1. 에이전트 리서치
2. remotion-slides 스킬 체크
3. markdowns 폴더 하위에 주제 폴더 만들기
4. 슬라이드 초안 생성
5. 프레젠터나 에디터에서 결과물 체크
6. 마크다운 수정도 OK
