# [어떻게 동작하는가?] 마크다운 기반 렌더링

- 마크다운 중심으로 동작하는 시스템
- 에이전트로 마크다운 초안을 만들고, 다듬거나
- 에디터에서 마무리

---

# [COMPONENT] 템플릿을 스킬화해서 사용합니다

![](04-COMPONENT-a.png)

- 스킬화한 템플릿 10종 제공
- 스킬을 활용해서 마크다운을 작성하라고 '에이전트'에게 지시

---

# [주요 기능] 프레젠터

- 프레젠터로 발표합니다.
- [F]를 누르면 전체화면이 됩니다.
- [좌], [우] 키로 슬라이드를 이동합니다.
- Remotion의 애니메이션 효과를 프레젠터에서 사용할 수 있습니다.

---

# [주요 기능] 에디터

- 에디터 상태에서 수정할 수 있습니다.
- 템플릿 규칙에 의해 사용해야 합니다.
- 예를 들어 이미지를 허용하는 슬라이드 템플릿에서만 이미지를 넣을 수 있습니다.

---

# [이렇게 사용하세요] 추천 워크 프로세스

1. 에이전트 리서치
2. remotion-slides 스킬 체크
3. markdowns 폴더 하위에 주제 폴더 만들기
4. 슬라이드 초안 생성
5. 프레젠터나 에디터에서 결과물 체크
6. 마크다운 수정도 OK

---
